# OpenStar / StarCell

Welcome to **OpenStar** — an open, community-driven game engine and fantasy RPG playground built on the **StarCell** demo game. Whether you are here to build your dream game, learn how game systems work, or contribute to a growing ecosystem, we are thrilled to have you!

**StarCell** is the included demo: a real-time procedurally generated fantasy world with NPC societies, dungeon exploration, crafting, combat, quests, and an emergent simulation driven entirely by simple rules.

---

## What Is StarCell?

StarCell is a top-down grid-based fantasy game where:
- A procedurally generated world fills with NPC societies — farmers, miners, traders, warriors, kings — each pursuing their own goals and reacting to each other
- Dungeons, caves, and structures generate dynamically and evolve over time
- Combat, crafting, quests, magic, followers, and a faction system layer on top of a living simulation
- Everything runs on cellular automata and probability — the world changes while you play and while you are gone

It is designed as both a playable game and a learning resource for game design.

## Our Unique "Open Core" Model

We want this project to be as accessible as possible while ensuring the core system continually improves for everyone. We operate on an **Open Core with Reciprocity** model.

What does this mean for you?
* **Build & Sell:** You are totally free to use this project to build and sell your own commercial games or applications. No mandatory royalties or profit-sharing.
* **Keep Your IP:** Your original assets, story, and unique branding belong 100% to you.
* **Share the Upgrades:** If you modify or improve the *base project code*, we require that you grant us the right to roll those improvements back into the main repository for the whole community to enjoy.
* **Official Branding:** Want to use our trademark, get official ecosystem integration, or be promoted on our channels? Check out our [Community Support Program (CSP)](commercial_use.md).

## Download & Run

[![Download StarCell](https://img.shields.io/badge/Download-StarCell-blue?style=for-the-badge)](https://github.com/qcruz/starcell/archive/refs/heads/main.zip)

### macOS

1. Download and unzip the file above
2. Double-click **`Launch StarCell.command`** inside the folder
3. If macOS says it can't verify the app, go to **System Settings → Privacy & Security** and click **Open Anyway**

### Windows / Linux

1. Download and unzip the file above
2. Install Python 3 if you do not have it: [python.org](https://www.python.org/downloads/)
3. Install pygame: `pip install pygame`
4. Run: `python main.py`

> The game requires Python 3.9+ and pygame.
